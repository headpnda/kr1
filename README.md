# Мой учебный проект

## Description
Этот проект представляет собой программу на языке C, которая выводит информацию о студенте и контрольной работе. Программа демонстрирует базовые навыки работы с языком C и системой контроля версий Git.

## Installation
1. Клонируйте репозиторий
2. Установите компилятор GCC
3. Перейдите в директорию src

## Usage
Программа выводит информацию о студенте и контрольной работе.

```bash
gcc src/main.c -o program
./program
