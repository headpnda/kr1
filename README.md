# Мой учебный проект

## Description
Этот проект представляет собой консольное приложение на C. Программа выводит информацию о студенте и выполненной контрольной работе.

## Installation
1. Клонируйте репозиторий
2. Установите компилятор GCC
3. Перейдите в папку src

## Usage
Для компиляции и запуска программы выполните:
